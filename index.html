GUIDE OFFICIEL • DIRECT INSTALL

Installer Delta sur iOS & Android

Méthodes officieuses mais fiables. Suivez chaque étape avec les images.

deltaexploits.gg

iOS

Direct Install (UNOFFICIAL)
Ouvrir le site & Onglet iOS
Cliquer sur l'onglet iOS
  • Ouvrez Safari (obligatoire sur iPhone/iPad).
  • Allez sur deltaexploits.gg
  • Cliquez sur « ONGLET iOS » (cf image).
Direct Install (UNOFFICIAL)
Première pression : Direct Install (UNOFFICIAL)
  • Appuyez sur « Direct Install (UNOFFICIAL) ».
  • Une nouvelle fenêtre va s'ouvrir.
Deuxième clic : Direct Install
Cliquez à nouveau sur Direct Install
  • Nouvelle fenêtre : « Direct Install ».
  • Cliquez une seconde fois.
  • Le profil se télécharge → retour à l'accueil.
✅ Dernière étape iOS :

• Allez dans Réglages → Général → VPN et gestion de l'appareil.
• Appuyez sur le profil Delta → Installer (entrez votre code).
• Autorisez le certificat (Trust / Autoriser).
• Redémarrez l'iPhone/iPad puis lancez Delta.

Android

APK officiel
Onglet Android
Cliquer sur l'onglet Android
  • Ouvrez Chrome (ou n'importe quel navigateur).
  • Allez sur deltaexploits.gg
  • Cliquez sur « ONGLET ANDROID ».
Download for Android
Bouton Download for Android
  • Appuyez sur « Download for Android ».
  • Le fichier .APK est téléchargé.
  • Cliquez sur les trois petits points du navigateur → allez dans « Téléchargements ».
  • Cliquez sur le fichier Delta → Installer.
⚠️ TRÈS IMPORTANT (Android) :

Avant de lancer Delta, vous DEVEZ désinstaller Roblox original.

➜ Paramètres → Applications → Roblox → Désinstaller.
➜ Ensuite seulement, lancez Delta. Le jeu fonctionnera via Delta.
➜ Ne gardez pas deux versions de Roblox.
✅ Lancer Delta :

• Une fois l'APK installé, retournez sur l'écran d'accueil.
• Lancez l'application Delta.
• Ouvrez Roblox depuis l'écran d'accueil Delta → connectez-vous et profitez.
